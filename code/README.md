# Code Repositories

## TransformerLens
- **URL:** https://github.com/TransformerLensOrg/TransformerLens
- **Purpose:** Library for mechanistic interpretability of GPT-2, GPT-3, and other LLMs. Essential for probing activations and residuals.
- **Key usage:** Accessing residual stream vectors to test if "washing machine" is a linear composition.

## concept-erasure
- **URL:** https://github.com/EleutherAI/concept-erasure
- **Purpose:** Tools for erasing concepts from representations. Useful to test if erasing "washing" affects "washing machine".
